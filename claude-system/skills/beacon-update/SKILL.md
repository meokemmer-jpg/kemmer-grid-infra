# /beacon-update [CRUX-MK]

## Wann nutzen
Nach jeder wichtigen Aenderung im branch-hub. Andere Branches pruefen BEACON.md beim Bootstrap (50 Tokens). Wenn BEACON neuer als ihr letzter Check: lesen sie BULLETIN.

## Schritte
1. Oeffne `G:/Meine Ablage/Claude-Knowledge-System/branch-hub/BEACON.md`
2. Ersetze die EINE Zeile mit: `LETZTE AENDERUNG: [ISO-Timestamp] | VON: [Branch] | THEMA: [1 Satz]`
3. Fertig. Keine weiteren Aktionen noetig.

## Beispiel
```
LETZTE AENDERUNG: 2026-04-08T10:00 | VON: Alpha | THEMA: MYZ-31 Code deployed + Knowledge-Diff geschrieben
```

## Kosten
50 Tokens pro Leser. Bei 5 Branches: 250 Tokens pro Check-Zyklus. Vernachlaessigbar.
